# 安全政策

## 支持版本

目前只维护最新的 GitHub Release。建议始终使用最新版本，或从最新源码自行打包。

## 报告安全问题

如果你发现与本项目有关的安全问题，请先创建一个不包含敏感细节的 GitHub Issue，并在标题中标注 `[Security]`。维护者会在确认后继续沟通复现步骤和影响范围。

请不要在公开 Issue 中直接发布可利用的细节、恶意样本或会影响其他用户的数据。

## 运行提示

Release 中的 Windows EXE 目前没有代码签名，首次运行时可能触发 Windows SmartScreen 提醒。你也可以选择从源码运行或自行打包。
