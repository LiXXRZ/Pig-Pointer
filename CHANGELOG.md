# 更新日志

本项目遵循轻量的人工更新日志格式。正式发布时建议同步创建 GitHub Release。

## 未发布

- 上传素材时复用 renderer 首次读取结果，避免预处理二次读取文件。
- 自定义资源区分静态图片和动画素材，多选时动画速度、触发概率和循环选项只作用于动画素材。
- 动画素材 renderer 缓存新增优先级，连续播放、正在播放或当前选中的动画资源会保留更多缓存。
- 运行中新增帧预算记录、动画分帧预热、智能档绘制超预算降频和透明合成画布复用。
- 多资源密集碰撞时新增碰撞降频和近邻候选限制。
- 自定义上传素材新增尺寸、帧数、文件大小和帧像素量预处理，用于更细的性能提醒。
- 多资源碰撞新增空间分桶起步优化，资源较多时减少无效碰撞检查。
- 将原“低性能”档替换为“智能性能”档，静止时自动降频并减少无意义重绘。
- 改进 README 顶部项目状态展示、Release 下载入口和使用注意。
- 将打包 EXE 从源码仓库移出，改为通过 GitHub Release 分发。
- 新增安全政策说明。
- 新增 README 界面预览图和默认猪猪 GIF 展示。
- 改进自定义资源连接点校准：支持预览拖动和滑块微调。
- 自定义资源支持 Ctrl / Shift 多选批量修改参数。
- 新增 GIF 连续播放选项。
- 新增独立绳子粗细设置。
- “都连到鼠标”模式下，多根绳子共用同一个鼠标锚点。
- 改进自定义模式状态提示排版。
- 禁用窗口最大化按钮，保留受限范围内的手动拉伸。

## 0.1.0

- 初始可用版本。
- 支持默认猪猪动画、系统托盘、透明置顶显示、参数保存。
- 支持自定义图片 / GIF、多资源碰撞、资源预设、保存目录迁移。
